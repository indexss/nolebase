# 简答题
1. 为什么要全面改革开放 / 深化改革
	1. 生产力是生产关系的基础，生产力决定了生产关系。生产力的性质决定了生产关系的性质，生产力的变化引起生产关系的变化，决定生产关系的变化方向
	2. 生产关系也会反作用于生产力。生产关系落后于生产力，会阻碍生产力的发展，若生产关系远超生产力的需要时，也会阻碍生产力的
	3. 所以说，生产力和生产关系是辩证统一的。生产关系一定要适合生产力的状况。当生产关系不能适应生产力的发展需要时，就要变革旧的生产关系，以适应发展生产力的需要。所以要全面深化改革，推动生产关系和生产力更好地适应。
	4. 经济基础决定上层建筑，经济基础是上层建筑以产生，存在的物质基础；上层建筑是经济基础得以确立其统治地位的条件，经济基础的变更必然引起上层建筑的变革，决定变革的方向
	5. 上层建筑对经济基础有反作用。上层建筑可以巩固经济基础使其成为统治地位，上层建筑和经济基础构成了二元矛盾运动。
	6. 所以，经济基础和上层建筑是辩证统一的，上层建筑一定要适合经济基础。经济基础决定上层建筑的发展方向，决定其变革方向，而上层建筑也会反作用于经济基础的性质和客观要求。进一步深化改革，使得经济基础和上层建筑相适应，从而使生产关系更适应生产力从而解放和发展生产力。
2. 马克思主义中国化，时代化的内涵与过程（中特的发展过程，中特就是中国化时代化的马）
	1. 内涵就是指将马克思主义与中国的实际情况相结合，具体来说，就是把马克思主义的基本原理与中国的革命，建设，改革实践，中国历史，文化结合起来，使马克思主义在中国具体化。
		1. 用马克思主义解决中国的革命，建设，改革的实际问题
		2. 把中国革命，建设和改革的实践经验和历史经验升华为理论，形成中国化马克思主义的理论成果
		3. 将马克思主义根植于中国优秀文化的土壤里，和中华民族的特点相融合，用人民喜闻乐见的民族形式表现处理
	2. 过程就是把五个领导人拿出来
		1. 新民主主义革命时期：以毛为代表的共产党人，把马列主义基本原理与中国具体情况相结合，创立了毛泽东思想，为夺取新民主主义革命胜利指明了正确方向
		2. 社会主义革命和建设时期，毛把马列主义基本原理和中国实际进行了第二次结合，为在新的历史时期开创中特社提供了宝贵经验，理论准备和物质基础
		3. 十一届三中全会之后，以邓为核心的共产党人，围绕什么是社会主义，如何建设社会主义的问题这一根本问题，借鉴了世界社会主义历史经验创立了邓小平理论
		4. 十三届四中全会后，以江为核心的共产党人，加深了对什么是社会主义，如何建设社会主义，以及建设社么样的党，怎么建设党的认识，形成了三个代表重要思想
		5. 党的十六大之后，以胡为代表的共产党人深刻认识回答了新形势下实现什么样的发展，怎样发展等重大问题，形成了科学发展观，
		6. 十八大以来，中特社进入新时代，以习为核心的共产党人坚持把马列基本原理和中国具体情况相结合，同中国优秀传统文化相结合，充分利用建党以来的历史经验，从新的实际出发，创立了习近平新时代中国特色社会主义思想，实现了马克思主义新的飞跃

4. 新民主主义总路线和纲领，意义，三大法宝
	1. 总路线：无产阶级领导的，人民大众的，反对帝国主义，封建主义和官僚资本主义的革命。（无产阶级是基本动力领导力量，农民是主力军，城市小资产阶级是无产阶级的可靠同盟，民族资产阶级也是动力之一。）
	2. 纲领：
		1. 政治纲领
			1. 在中国建立一个以无产阶级领导的，工农联盟为基础的，反对帝国主义，封建主义，官僚资产阶级的各阶级联合专政的民主共和国，即新民主主义共和国
			2. 国体：各革命阶级联合专政。政体：民主集中制。这就是新民主主义的政治，新民主主义共和国。
		2. 经济纲领
			1. 没收封建地主阶级的土地归农民所有，没收蒋为首的垄断资本归新民主主义国家所有，保护民族工商业（特色）。
			2. 新民主主义国家有五种经济成分，社会主义性质的国营经济，半社会主义的合作设计经济，私人资本主义，个体经济，国家资本主义
		3. 文化纲领
			1. 无产阶级领导的反帝反封建的文化，即民族的，科学的，大众的文化。
	3. 三大法宝：
		1. 统一战线：巩固工农联盟，正确对待资产阶级，区别对待
		2. 武装斗争：坚持军队的绝对领导，开展革命政治工作，坚持正确的战略战术
		3. 党的建设：思想建设要放在党建首位，重视党组织建设，党的作风建设
	4. 意义：
		1. 新民主主义的伟大胜利，改变世界格局，鼓舞了全世界被压迫民族和被压迫人民争取解放的斗争，增强了他们反帝反封建的信心，增强了世界人民争取和平的力量。
		2. 运用马克思主义立场解决了中国的实际问题，对马克思主义中国化时代化有方法论意义。
5. 社会主义改造的内容，可能性和必然性，经验，历史意义
	1. 内容：一化三改，社会主义工业化，个体农业，手工业（互助合作，自愿互利，手工业改造为社会主义经济），资本主义工商业（初级形式的国家资本主义->个别企业的公私合营->全行业公私合营）的社会主义改造，鸟的主体和两翼。
	2. 可能和必然性：
		1. 社会主义性质的国营经济力量相对强大，是国家工业化的主要基础，可以用来投资搞大型工业建设。
		2. 资本主义经济力量弱小，发展困难，不能成为中国工业起飞的基础
		3. 对个体农业手工业进行社会主义改造，是保证工业腾飞，国家工业化的基础。当时已经有了农业合作的基础。
		4. 国际环境促使中国选择社会主义
	3. 经验：
		1. 并举。社会主义工业化和社会主义改造要并举。
		2. 积极引导，逐步过渡
		3. 和平改造
	4. 历史意义
		1. 发展了国民经济，得到了广大人民的拥护
		2. 建立了社会主义经济制度，中国从新民主主义社会变成了社会主义初级阶段
		3. 将马列主义与中国实际情况结合，丰富了马克思主义
		4. 解放了我国的生产力，促进了生产力发展
