# Face Description with Local Binary Patterns: Application to Face Recognition

摘要：

本文（引用本文）研究了人脸识别中人脸特征表示的问题。该工作收到了LBP（引用原文）在纹理描述应用的启发，在人脸图像的多个局部区域分别上使用LBP算子形成空间增强的全局特征向量用于人脸识别，在FERET数据集上达到了最优性能。

相关工作：

1. 整体方法：如主成分分析 (PCA) [2], 、线性判别分析 (LDA) [3], 以及最近的 2D PCA [4]，对于姿态和光照等鲁棒性较差
2. 局部方法：Pentland 5 对局部人脸区域使用PCA，LFA 6 中采用局部空间支持核来提取局部信息， EBGM 7中对面部标志点处使用Gabor滤波器提取局部特征，用图结构形成空间关系。Heisele 8 证明局部方法在旋转不变性上优于整体方法
3. LPF 9 在目标识别领域效果好，但这种bag-of-keypoints由于缺少空间信息，不适合面部特征任务。

方法：
本工作假设人脸图像可视为微模式的组合，而LBP算子捕捉这些微模式。作者采用了LBP算子在局部捕捉特征。作者首先介绍了基本LBP算子，在3x3的窗口内以中心像素为阈值，若领居大于阈值则为1，小于阈值则为0，得到一个用于描述该区域特征的二进制串。然而，由于这种基本方法智能覆盖固定半径的区域，所以作者引入了圆形LBP算子，并对未落在像素中心的点进行双线性插值。然而，这样会存在采样模式过多的问题，为了解决这一问题，作者还引入了**uniform LBP 模式**，限制跳变次数为2，降低了模式数量，减少了高频噪声对模式的影响。

在局部特征与整体特征的提取方面，作者认为，局部特征和混合方法在前人实验结果上优于整体特征。另外，单个LBP直方图会把空间位置平均掉，而人脸识别汇总空间位置十分重要，所以作者采用了分块的方式保留了位置信息。具体来说，人脸图像被分为多个区域，每个区域用上述的LBP算子得到直方图，最后将所有直方图按空间顺序连接得到空间增强直方图，即可得到面部特征。

在距离度量方面，作者假设一些面部特征比其它特征在面部辨别任务上更重要，所以作者定义了加权卡方距离来度量特征相似度。

数据集，实验，结果：
作者采用了FERET人脸数据库，含有1,199个个体在不同表情，光照，姿态，时间下的14,051张灰度图像。
评估方面，本工作采用了CSU Face Identification Evaluation System 15，并使用rank 1识别率，the mean recognition rate with a 95 percent confidence interval以及Permutation Test作为指标。
1. 参数选择实验：作者通过测试不同LBP参数找到了最佳参数组合，并根据在数据集上的识别率找到了卡方距离的权重。作者发现LBP的参数选择十分鲁棒。
2. 算子选择实验：作者在分块区域直方图的情况下，替换不同Local Descriptors，结果发现各方法对表情变化鲁棒，但只有LBP能够应对光照变化。LBP在texton在时间变化上也有最佳性能。作者认为LBP的优势来自于对单调灰度变化的容忍性。
3. 对比实验：作者提出的weighted LBP的识别率优于其他所有对比方法。
4. Localization Error Robustness experiment：作者为人脸加入随机平移误差进行实验，结果表明，在误差较小或没有误差时，小区域LBP效果更好，在定位误差变大后，大区域效果更好，且LBP性能下降速度低于PCA。

优点：
本位提出了一种采用局部LBP的方法描述人脸特征，显式保留了局部与空间结构。

本方法思路自然，在观察到人脸识别任务中使用局部特征优于整体特征，保留空间特征效果更佳，LBP算子性能优秀三个洞察后自然将三者结合，为读者提供了创新方法的思路。

本文实验设计十分系统，在多场景下评判了方法的有效性，也与多种方法进行了对比。另外，作者也详细解释了方法参数的选择，并有实验支撑。

缺点：
具有时代局限性，采用基于经验的划分方式和权重，难以证明在生活中的有效性。若能让系统学习划分方式以及权重间的关系，预计会有更好结果。

仅在单一数据集上实验，未能使用更多数据集，没有在非受控环境下实验。

思考：
作者将多种前人的工作思路组合，得到的新方法效果很好，这启发我们要尝试结合多种方法的优点，也许这种简单的方式能够有很好的结果。

另外，本文的区域划分方法与ViT（引）的划分十分相似，然而，优于采用attention机制，ViT能够克服经验主义，自适应寻找加权机制。这启发我们在遇到加权，或者patch处理的旧方法，不妨尝试加入attention。
